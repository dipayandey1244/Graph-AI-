# Contributing

Thanks for contributing to Graph-AI-!

## How to Contribute

1. Fork the repository and create a feature branch.
2. Keep changes focused and well-scoped.
3. Add or update docs for any behavior or structure changes.
4. Run the quality checks before opening a PR.

## Development Workflow

```bash
make install
make lint
make format
make docs
make run-demo
```

## Pull Request Guidelines

- Include a clear summary of what changed and why.
- Reference related issues if applicable.
- Prefer small, reviewable pull requests.

## Code Style

- Use clear names and concise module-level documentation.
- Keep examples runnable and easy to follow.
- Favor incremental complexity from easy to advanced.
